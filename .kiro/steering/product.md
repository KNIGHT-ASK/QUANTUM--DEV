# Product Overview

Roo Code is an AI-powered development assistant VS Code extension that helps developers write, refactor, debug, and understand code through natural language interactions.

## Core Capabilities

- **Code Generation**: Create code from natural language descriptions and specifications
- **Multiple Modes**: Adapt to different workflows (Code, Architect, Ask, Debug, Custom)
- **Refactoring & Debugging**: Improve and fix existing code
- **Documentation**: Write and update project documentation
- **Codebase Understanding**: Answer questions about the codebase
- **Task Automation**: Automate repetitive development tasks
- **MCP Integration**: Utilize Model Context Protocol servers for extended functionality

## Key Features

- **Modes System**: Different operational modes for specific tasks (Code, Architect, Ask, Debug, Custom)
- **Roomote Control**: Remote task control for local VS Code instances
- **Multi-provider AI**: Support for various AI providers (Anthropic, OpenAI, Google, AWS Bedrock, Azure, Ollama, etc.)
- **Codebase Indexing**: Semantic search and understanding of project code
- **Checkpoints**: Save and restore conversation states
- **Todo Lists**: Track and manage development tasks
- **i18n Support**: Available in 17+ languages

## Target Users

Professional developers and teams looking for an AI coding assistant that integrates directly into their VS Code workflow with autonomous capabilities and multi-model support.
